# 🐉 SHEvyon — סימולציית תיק השקעות (גרסה עצמאית)

אפליקציית משחק אישית לסימולציית השקעות בשוק ההון לשנים 2020–2024.  
פותחה עבור **SHEvyon** כלי חינוכי להדרכות ואירועים.

🔗 **Live:** [shevyon-solo.netlify.app](https://shevyon-solo.netlify.app)

---

## 📁 מבנה הפרויקט

```
shevyon-solo/
├── index.html          # מבנה המסכים
├── css/
│   └── styles.css      # עיצוב (מיתוג SHEvyon — סגול/לבן)
└── js/
    ├── data.js         # נתוני שנים, נכסים, קבועים
    └── app.js          # לוגיקת המשחק
```

---

## 🎮 זרימת המשחק

1. **מסך פתיחה** — הוראות + דיסקליימר → "בואי נעשה ניסיון!"
2. **שנת ניסיון** — 2020, ללא ספירת עמלות, ללא שמירה
3. **סיכום ניסיון** → "יאללה מתחילות לשחק!"
4. **משחק אמיתי** — שנים 2020–2024, כל שנה:
   - בחירת אלוקציה (סליידרים, 180 שניות)
   - שינוי תיק עולה 2% עמלה
   - סיכום שנה עם תובנה
5. **תוצאות סופיות** — שווי, תשואה, ביצועים לפי שנה, עמלות
6. **מסך שיווקי** → קישור לאתר SHEvyon
